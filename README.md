# ensambliaAPI
🎵 Ensamblia - Red Social para Músicos Ensamblia conecta músicos, bandas y amantes de la música. Plataforma con perfiles personalizados, anuncios musicales, chats, grupos y multimedia. Base de datos PostgreSQL optimizada con 500+ usuarios de prueba. Creada por Maite, Marga, Raquel y Luis. 🎸🎹🥁
